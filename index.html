MY SKILL

HTML

HTML stands for HyperText Markup Language. It's the standard language used to create and design web pages. HTML is the backbone of any web page, providing the structure and layout by using various tags and elements to define the different parts of a document.

CSS

the presentation and formatting of a document written in a markup language like HTML. In simpler terms, CSS is used to control the look and feel of a web page, determining how elements are displayed on a screen or other media.

JAVASCRIPT

programming language used to make web pages interactive and dynamic. Unlike HTML, which is used to create content structure, and CSS, which is used to style the appearance of a page, JavaScript provides the ability to control the behavior of a web page.
